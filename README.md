# LittleLeafy

A cute, browser-based plant pot generator for 3D printing.

Customize a pot, preview it in 3D, and download a printable STL or a ZIP bundle with the STL and selected settings. Everything runs locally in the browser with no login, backend, database, or paid APIs.

## Features

- Live 3D pot preview with mouse controls
- Custom height, diameters, wall thickness, rim, drainage, and pattern style
- Editable planter templates for nicer starting shapes
- Client-side STL export
- ZIP export with `README.txt`
- Mobile-friendly Next.js + Tailwind UI

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```
